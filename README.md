# 2040-mini-core


Mini Core similar to the Arduino one for the [2040](https://datasheets.raspberrypi.com/rp2040/rp2040-datasheet.pdf) microcontroller 


## How to build

```
mkdir build
cd build
cmake ../
make
```

then just upload ```2040_Core.uf2``` to your 2040
